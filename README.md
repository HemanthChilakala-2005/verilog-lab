# verilog-lab

## Starting GitHub Codespaces

If you're using GitHub Codespaces to work on this lab, follow these steps:

1. Open the repository on GitHub.
2. Click the **Code** button.
3. Select the **Codespaces** tab.
4. Click **Create codespace on main** (or start an existing Codespace if one already exists).
5. Wait for the environment to finish setting up. This may take a few minutes the first time.
6. Once the Codespace opens in your browser, open the integrated terminal and begin working on the lab.

You're now ready to edit files, run commands, and complete the Verilog lab directly from your browser.
